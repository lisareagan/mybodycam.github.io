MyBodyCam SMS Notifications

MyBodyCam allows users to add trusted contacts (“Safety Circle members”) to receive safety-related SMS notifications.

By checking the box above, you authorize MyBodyCam to send you text messages using an automatic telephone dialing system or other automated technology. No marketing messages.

Please enter a mobile phone number and check the consent box.

Message frequency varies based on user activity (typically 4–6 messages per month). Message and data rates may apply.

Reply STOP at any time to opt out. Reply HELP for assistance.

Consent to receive SMS messages is not required to use MyBodyCam and is separate from acceptance of the Privacy Policy and Terms.
